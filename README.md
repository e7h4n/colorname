# Color Name

To get rid of naming hell.

## Install

```bash
npm install -g colorname
```

## Usage

```bash
$ colorname #82A1B7 #FFFFFF #E7EEF3 #5A5E61
#82A1B7: paleBlue(#7AB8CC)
#FFFFFF: white(#FFFFFF)
#E7EEF3: lavender(#E6E6FA)
#5A5E61: ironGray(#625B57)
```

Export less format:

```bash
$ colorname --less #82A1B7 #FFFFFF #E7EEF3 #5A5E61
@paleBlue: #82A1B7;
@white: #FFFFFF;
@lavender: #E7EEF3;
@ironGray: #5A5E61;
```
